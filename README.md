## csv-jsonify ![NPM version](https://img.shields.io/npm/v/csv-jsonify.svg?style=flat) 

a jsonify cli tool to convert csv to json

### Installation
```bash
$ npm install csv-jsonify
```

### Example
```js
var csvJsonify = require('csv-jsonify');
```
